# LayoutContent

The content area of the layout component.

## Props:

- None

## Slots:

- Default slot: The content to be displayed within the layout content area.

## Breadcrumbs:

It also usually houses the Breadcrumbs component at the top of the content

## Usage

```
<LayoutContent>
 <Breadcrumbs/>
  <!-- Content goes here -->
 </LayoutContent>
```
